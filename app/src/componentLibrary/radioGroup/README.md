## **Radio button list**

### Props:

- **value**: _bool_, optional, default = false
- **options**: _array_, optional, default = []
- **disabled**: _bool_, optional, default = false
- **variant**: _sting_, optional, default = "light"
- **className**: _string_, optional, default = ""
- **mobileDisabled**: _bool_, optional, default = false

### Events:

- **onFocus**
- **onBlur**
- **onChange**

### Variants:

Radio button list can be used in two variants: **light** for light background and **dark** for dark background.
